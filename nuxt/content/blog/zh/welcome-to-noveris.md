# 欢迎来到 Noveris Blog

我们很高兴地宣布 Noveris Blog 正式上线！这是一个专注于 AI、Web 开发和技术洞察的技术博客。

## 我们的技术栈

Noveris Blog 采用了现代化的技术栈构建：

- **Nuxt 4** - Vue.js 的全栈框架
- **Nuxt Content** - Git-based 内容管理
- **TypeScript** - 类型安全的 JavaScript
- **Meilisearch** - 高性能搜索引擎
- **Giscus** - 基于 GitHub Discussions 的评论系统

## 内容规划

我们计划涵盖以下主题：

1. **人工智能** - 机器学习、深度学习、LLM 应用
2. **Web 开发** - 前端框架、后端技术、DevOps
3. **技术洞察** - 行业趋势、最佳实践、经验分享

## 加入我们

欢迎通过以下方式与我们互动：

- 在文章下方留言评论
- 通过 GitHub 提交贡献
- 订阅我们的 Newsletter

敬请期待更多精彩内容！
