# Welcome to Noveris Blog

We are excited to announce the official launch of Noveris Blog! This is a tech blog focused on AI, Web Development, and Technology Insights.

## Our Tech Stack

Noveris Blog is built with a modern tech stack:

- **Nuxt 4** - Full-stack Vue.js framework
- **Nuxt Content** - Git-based content management
- **TypeScript** - Type-safe JavaScript
- **Meilisearch** - High-performance search engine
- **Giscus** - GitHub Discussions-powered comments

## Content Plan

We plan to cover the following topics:

1. **Artificial Intelligence** - Machine Learning, Deep Learning, LLM Applications
2. **Web Development** - Frontend Frameworks, Backend Technologies, DevOps
3. **Tech Insights** - Industry Trends, Best Practices, Experience Sharing
